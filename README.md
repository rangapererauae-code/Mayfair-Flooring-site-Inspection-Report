# Mayfair Flooring - Site Inspection Report

An interactive, iOS-optimized inspection report tool for flooring professionals.

## Features

- 📱 Fully responsive & iOS optimized
- 📸 Photo upload with comments per image
- 🖨️ Print / PDF export support
- 📋 Complete inspection form with:
  - Project details
  - Flooring scope (SPC, LVT, Wood, Carpet)
  - Site conditions
  - Structural variations
  - Digital sign-off
- 💾 All data stays client-side

## Live Demo

[View the inspection tool]https://rangapererauae-code.github.io/Mayfair-Flooring-site-Inspection-Report/

## Usage

1. Fill in project information
2. Take/upload photos with comments
3. Complete site assessment
4. Print or save as PDF
5. Clear form for next inspection

## Browser Support

Works on all modern browsers, optimized for Safari on iOS.
